📖 使用说明

启动程序

解压后运行 ./start.sh (Linux/macOS) 或双击 start.bat (Windows)

创建管理员账号

首次访问会提示创建管理员账号，输入用户名和密码即可

打开数据库

点击"浏览服务器文件"选择数据库，或"新建数据库"创建新的数据库

管理数据

点击左侧表名查看数据，双击单元格编辑，点击"新建表"创建新表

常用命令

查看版本 ./sqlite-manager version
重置密码 ./sqlite-manager reset-password
指定端口 ./sqlite-manager -port 3000
查看帮助 ./sqlite-manager help

Docker 运行

运行容器 docker run -p 8903:8903 -v ./data:/data techfunways/sqlite-manage:v1.1.0-amd64
加载镜像 docker load < xxx.docker.tar.gz
